# NIXX - Neon Edition 🏍️⚡

**NIXX - Neon Edition** é um jogo estilo TRON desenvolvido em Java, onde o objetivo é sobreviver e forçar seus oponentes a colidirem com os rastros de luz deixados pelas motos.

## 🚀 Funcionalidades

- **Múltiplos Modos de Jogo:** PvE, PvP e CO-OP.
- **Gráficos Estilizados:** Interface moderna com tema neon.
- **IA Desafiadora:** Inimigos que ficam mais inteligentes a cada nível no modo PvE.
- **Suporte para Dois Jogadores:** Jogue localmente com um amigo.

## 🕹️ Como Jogar

O objetivo é simples: não bata nas paredes, no seu próprio rastro ou no rastro dos seus oponentes.

### Controles

| Jogador | Cor | Teclas |
| :--- | :--- | :--- |
| **Jogador 1** | Azul 🔵 | Setas Direcionais (↑ ↓ ← →) |
| **Jogador 2** | Branco ⚪ | Teclas W, A, S, D |

### Modos de Jogo

- **PvE (Player vs Environment):** Sobreviva contra IAs em níveis progressivos.
- **PvP (Player vs Player):** Um duelo direto entre dois jogadores humanos.
- **CO-OP (Cooperativo):** Trabalhe em conjunto para eliminar as ondas de inimigos.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java
- **Interface Gráfica:** Java Swing / AWT
- **Arquitetura:** Orientada a Objetos
